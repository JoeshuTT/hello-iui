# iui - 一个基于 uni-app 的 UI 组件库（兼容NVUE）

## 介绍

`iui` 是一款基于uni-app，面向移动端，兼顾NVUE的UI组件库，适用于跨端开发。

目标

- 打造高性能表现，高质量代码的组件库
- 完成上手简单，易于使用的目标 
- 在不牺牲各平台特色的情况下，处理跨端
